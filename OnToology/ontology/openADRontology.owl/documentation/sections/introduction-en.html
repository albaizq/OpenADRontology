Introduction back to ToC

Open Automated Demand Response (OpenADR) provides a non-proprietary, open standardized DR interface that allows electricity providers to communicate DR signals directly to existing customers using a common language and existing communications such as the Internet. This document describes the OpenADR ontology, which aims at modelling the data exchanged by the users of the OpenADR protocol.
